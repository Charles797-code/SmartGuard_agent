# SmartGuard 智能反诈助手 - 用户手册

## 简介

SmartGuard 是一款基于多模态 AI 技术的智能反诈助手，为滨江区浙工大网络空间安全创新研究院提供智能化反诈解决方案。系统支持文本、图片、语音等多种输入方式，实时分析对话内容，识别潜在诈骗风险，并提供分级预警和监护人联动功能。

---

## 主要功能概览

| 功能 | 说明 |
|------|------|
| 风险分析 | 发送可疑内容（文本/图片/语音），AI 智能判断是否诈骗 |
| 反诈助手 | 聊天咨询，了解各类诈骗手法和防范技巧 |
| 知识百科 | 学习 12 种常见诈骗类型的套路、案例和防范方法 |
| 预警中心 | 查看本人及被监护人的风险预警记录 |
| 监护人管理 | 添加/编辑紧急联系人，高风险时自动通知 |
| 举报功能 | 提交诈骗线索，帮助优化系统检测能力 |

## 功能详解

### 1. 风险分析模式

**用途**：分析可疑短信、来电内容、聊天记录是否存在诈骗风险。

**使用方法**：
1. 点击左侧「风险分析」按钮
2. 在输入框粘贴可疑内容（短信、聊天记录、网址等）
3. 可选：点击 📷 上传截图，点击 🎤 上传录音
4. 点击「分析」按钮发送

**支持输入**：
- 文本：粘贴可疑短信、聊天内容、网址等
- 图片：上传可疑截图（系统会提取图片中的文字）
- 音频：上传可疑录音（自动转写并分析）

**风险等级**：

| 等级 | 名称 | 颜色 | 说明 |
|------|------|------|------|
| 0 | 安全 | 绿色 | 正常交流，无明显风险 |
| 1 | 关注 | 蓝色 | 发现模糊风险信号，建议留意 |
| 2 | 警告 | 黄色 | 中度风险，需要警惕 |
| 3 | 危险 | 橙色 | 高风险，建议停止操作 |
| 4 | 紧急 | 红色 | 紧急风险，系统已通知监护人 |

---

### 2. 反诈助手模式

**用途**：像和朋友聊天一样咨询反诈问题，获得针对性的防骗建议。

**使用方法**：
1. 点击左侧「反诈助手」按钮
2. 在输入框输入问题（可以发送文字/图片/语音）
3. AI 会根据你的年龄、职业等信息提供个性化建议

**示例问题**：
- "朋友让我帮他转账，这是诈骗吗？"
- "网上有人说可以帮我消除征信记录，真的吗？"
- "我收到一条短信说我中大奖了，怎么判断是不是诈骗？"

**支持的诈骗类型**：

|------|---------|------|
| 冒充公检法诈骗 | 极高 | 假冒公安、检察院、法院人员 |
| 深度伪造诈骗（AI换脸/变声） | 极高 | 利用 AI 技术冒充熟人 |
| AI诈骗（绑架） | 高 | 合成子女声音谎称被绑架 |
| 投资理财诈骗 | 高 | 虚假投资平台、高收益诱惑 |
| 兼职刷单诈骗 | 高 | 刷单兼职、点赞返利 |
| 杀猪盘诈骗 | 高 | 婚恋诈骗、情感投资 |
| 购物退款诈骗 | 中 | 退款陷阱、双倍赔偿 |
| 医保诈骗 | 中 | 医保盗用、异地报销 |
| 虚假贷款诈骗 | 中 | 无抵押贷款、套路贷 |
| 虚假征信诈骗 | 低 | 征信修复、逾期洗白 |
| 游戏交易诈骗 | 低 | 游戏装备、账号交易 |
| 追星诈骗 | 低 | 粉丝群、明星福利 |

**使用方法**：
1. 点击左侧「知识百科」按钮
2. 查看诈骗类型卡片，点击任意类型查看详情
3. 详情页包含：诈骗套路步骤、典型案例、防范技巧、警示信号

### 4. 预警中心

**用途**：查看本人及被监护人的风险预警记录。

**功能**：
- 查看所有历史预警记录
- 按风险等级筛选（高/中/低）
- 查看预警详情和处理状态
- 标记已处理

**预警通知规则**：
- 风险等级 3-4：系统自动通知已添加的监护人
- 监护人收到通知后可在 3 秒内选择"确认"或"取消"

---

### 5. 监护人管理

**用途**：添加紧急联系人，系统在检测到高风险时自动通知监护人。

**添加监护人**：
1. 点击左侧「监护人」按钮
2. 点击「添加监护人」按钮
3. 填写信息：
   - 姓名
   - 手机号
   - 与你的关系（父亲/母亲/配偶/子女/其他）
   - 通知方式（短信/微信/APP推送）

**管理功能**：
- 编辑监护人信息
- 删除监护人
- 开启/关闭通知
- 查看通知历史

---

### 6. 举报功能

**用途**：提交发现的诈骗线索，帮助优化系统检测能力。

**举报类型**：
- 冒充公检法
- 投资理财诈骗
- 兼职刷单诈骗
- 杀猪盘诈骗
- AI 诈骗
- 虚假贷款
- 购物退款
- 其他

**举报信息**：
- 诈骗类型
- 诈骗描述（必填）
- 对方账号/电话
- 涉及金额（可选）
- 发生时间
- 上传证据截图（可选）

**举报状态**：
| 状态 | 说明 |
|------|------|
| 待处理 | 举报已提交，等待审核 |
| 已核实 | 举报内容已确认，进入学习库 |
| 已驳回 | 举报内容不符合条件 |

### 7. 学习中心

**用途**：查看系统从举报中学习的最新诈骗手法。

**功能**：
- 查看最新识别的新诈骗模式
- 了解系统进化学习成果
- 查看累计学习的知识数量

### 8. 用户设置

**用途**：管理个人资料和系统偏好。

**可设置项**：
- 昵称
- 年龄段（老年人/成年人/未成年人）
- 职业
- 风险偏好（保守/正常/激进）
- 预警风格（温和/正常/紧急）
- 是否独居
- 是否曾有被骗经历
- 防骗意识水平

**说明**：
- 年龄段会影响风险判断权重（老年人权重更高）
- 风险偏好会影响预警敏感度
- 防骗意识水平越高，误报可能越少

## 操作指南

### 首次使用

1. **设置用户画像**
   - 点击左侧「设置」
   - 填写年龄段、职业等信息
   - 点击「保存设置」

2. **添加监护人**（推荐）
   - 点击左侧「监护人」
   - 添加至少一位紧急联系人
   - 开启通知功能

3. **开始使用**
   - 选择「风险分析」或「反诈助手」模式
   - 发送可疑内容进行分析

### 日常使用

**分析可疑内容**：
```
风险分析 → 粘贴内容 → 上传图片/音频(可选) → 分析 → 查看结果
```

**咨询反诈问题**：
```
反诈助手 → 输入问题 → 发送 → 查看回答
```

**学习防骗知识**：
```
知识百科 → 选择诈骗类型 → 查看详情 → 学习防范技巧
```

## 风险等级说明

### 等级 0 - 安全（绿色）
正常交流，未检测到诈骗特征。

### 等级 1 - 关注（蓝色）
发现模糊风险信号，例如：
- 涉及金钱但金额不大
- 语气略有异常

**建议**：正常交流即可，如有疑虑可进一步咨询。

### 等级 2 - 警告（黄色）
检测到可疑特征，例如：
- 提及"安全账户"
- 要求转账到陌生账户
- 声称是官方机构

**建议**：提高警惕，核实对方身份，不要转账。

### 等级 3 - 危险（橙色）
高风险诈骗特征明显，例如：
- 频繁催促转账
- 威胁冻结账户
- 要求保密

**建议**：立即停止操作，不要转账，可向官方渠道核实。

### 等级 4 - 紧急（红色）
检测到典型诈骗话术，系统已通知监护人。

**建议**：保持冷静，拨打 96110 反诈热线咨询。

## 诈骗类型详解

### 冒充公检法诈骗
**特征**：假冒公安局、检察院、法院人员
**套路**：声称你涉嫌洗钱/贩毒 → 要求核查资金 → 转入"安全账户"
**关键词**：安全账户、资金核查、拘捕令、资产冻结

### AI诈骗（深度伪造）
**特征**：利用 AI 换脸、变声技术冒充熟人
**套路**：合成子女/朋友声音/视频 → 声称出事 → 要求紧急转账
**关键词**：出事了、急需用钱、别告诉别人、转账

### 投资理财诈骗
**特征**：虚假投资平台，高收益诱惑
**套路**：导师带单 → 小额盈利 → 大额投入 → 无法提现
**关键词**：稳赚不赔、高收益、内幕消息、导师带单

### 兼职刷单诈骗
**特征**：刷单兼职、点赞返利
**套路**：小任务返佣金 → 逐步加大金额 → 任务失败无法提现
**关键词**：刷单、日结、足不出户、连环任务

### 杀猪盘诈骗
**特征**：婚恋交友，建立感情后诱导投资
**套路**：网络邂逅 → 甜言蜜语 → 透露投资内幕 → 诱导转账
**关键词**：博彩平台、稳赚不赔、导师下注、一起赚钱

### 虚假贷款诈骗
**特征**：无抵押、快速放款
**套路**：下载贷款 APP → 显示额度 → 提示银行卡号错误 → 缴纳解冻费
**关键词**：无抵押、快速放款、解冻费、手续费

## 常见问题

### Q: 如何判断分析结果？

根据风险等级判断：
- 0-1级：正常交流，可继续
- 2级：需要警惕，核实对方身份
- 3-4级：立即停止操作，拨打 96110 咨询

### Q: 误报怎么办？

系统会持续学习优化。如遇误报可通过「举报」功能提交反馈，帮助改进检测能力。

### Q: 如何保护隐私？

- 对话数据本地存储，不上传云端
- 不在对话框中输入真实身份证号、银行卡密码等敏感信息
- 支持联系管理员删除个人数据

### Q: 启动后分析响应很慢？

首次启动时系统需要构建向量索引（约 1-2 分钟），之后每次启动秒级加载。如需加速可安装 FAISS：

```bash
pip install faiss-cpu
```

### Q: 图片分析支持哪些格式？

支持 JPG、PNG、GIF、WebP 等常见图片格式。

### Q: 语音分析支持哪些格式？

支持 MP3、WAV、AAC 等常见音频格式。

### Q: 监护人通知有什么延迟？

- 高风险检测后立即发送通知
- 监护人有 3 秒确认期
- 3 秒后自动通知，无需监护人确认
