# 用户使用指南

## 简介

SmartGuard是一款基于多模态AI技术的智能反诈助手，能够实时分析文本、语音、图像等多种输入，识别潜在的诈骗风险，并提供及时的预警和防护建议。

## 主要功能

### 1. 智能风险分析

- 支持文本、语音、图片等多种输入方式
- 实时分析对话内容，识别诈骗风险
- 覆盖12+种常见诈骗类型

### 2. 分级预警机制

| 等级 | 名称 | 说明 |
|------|------|------|
| 🟢 0 | 安全 | 正常交流，无风险 |
| 🟡 1 | 关注 | 发现模糊风险信号 |
| 🟠 2 | 警告 | 中度风险，需要警惕 |
| 🔴 3 | 危险 | 高风险，建议停止操作 |
| 🆘 4 | 紧急 | 紧急风险，已通知监护人 |

### 3. 监护人联动

- 添加紧急联系人
- 风险事件即时通知
- 远程干预支持

### 4. 个性化防护

- 基于年龄段的差异防护
- 用户画像动态调整
- 历史风险记录

## 使用方法

### 启动服务

```bash
# 方式一：交互模式
python main.py interactive

# 方式二：API服务模式
python main.py api --port 8000
```

### Web界面使用

1. 打开浏览器访问 http://localhost:8000
2. 在输入框中输入需要分析的内容
3. 点击"发送"按钮
4. 查看分析结果和风险等级

### API调用示例

#### 分析文本

```bash
curl -X POST http://localhost:8000/api/v1/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "text": "您好，我是公安局民警，需要您配合调查...",
    "modality": "text",
    "user_id": "user123"
  }'
```

#### 添加监护人

```bash
curl -X POST http://localhost:8000/api/v1/users/user123/guardians \
  -H "Content-Type: application/json" \
  -d '{
    "name": "张三",
    "phone": "13800138000",
    "relationship": "父亲"
  }'
```

#### 获取预警列表

```bash
curl http://localhost:8000/api/v1/users/user123/alerts
```

#### 生成安全报告

```bash
curl -X POST http://localhost:8000/api/v1/reports/generate \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "user123",
    "report_type": "daily"
  }'
```

## 设置用户画像

### 年龄段设置

- `elderly`: 老年人（60岁以上）
- `adult`: 成年人（18-60岁）
- `minor`: 未成年人（18岁以下）

### 风险偏好设置

- `conservative`: 保守（更严格的预警）
- `normal`: 正常
- `aggressive`: 激进（较宽松的预警）

## 常见问题

### Q: 如何判断分析结果？

根据风险等级判断：
- 0-1级：正常交流，可继续
- 2级：需要警惕，核实对方身份
- 3-4级：立即停止操作，报警处理

### Q: 误报怎么办？

系统会持续学习优化，如遇误报可通过反馈帮助改进。

### Q: 如何保护隐私？

- 对话数据本地存储
- 不上传敏感信息
- 支持数据删除
