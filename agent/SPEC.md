# 基于多模态的反诈智能助手 - 项目规格说明

## 1. 项目概述

### 1.1 项目名称
**SmartGuard** - 多模态反诈智能助手

### 1.2 核心目标
构建一个具备"感知-决策-干预-进化"能力的智能反诈助手，整合文本、语音、图像、视频等多模态信息，实现精准诈骗识别、实时预警和个性化防护。

### 1.3 目标用户
- 普通民众（老年人、儿童青少年、中青年）
- 监护人（家长、子女）
- 反诈宣传机构

## 3. 功能模块详细设计

### 3.1 模块一：多模态输入支持

#### 3.1.1 文本输入
- 支持聊天记录粘贴
- 支持短信截图OCR识别
- 支持富文本编辑

#### 3.1.2 音频输入
- 实时通话流处理
- 语音消息上传分析
- ASR语音转文本

#### 3.1.3 视觉输入
- 视频流实时分析
- 图片/截图上传
- 屏幕共享检测

#### 数据预处理流程
```
输入数据 → 格式校验 → 模态分离 → 特征提取 → 向量化 → 统一表示
```

### 3.2 模块二：智能识别与决策引擎

#### 3.2.1 意图识别 (Intent Recognition)
```
系统提示词设计:
- 角色设定：你是一位专业的反诈安全顾问
- 任务指令：分析用户输入，判断是否存在诈骗风险
- 决策逻辑：安全/低风险/中风险/高风险/紧急
- 少样本示例：提供典型诈骗对话示例
```

#### 3.2.2 知识库检索
- 反诈法律法规库
- 典型案例库 (10+种诈骗类型)
- 预警话术库
- 防护建议库

#### 3.2.3 诈骗类型识别 (≥10种)
| 编号 | 诈骗类型 | 特征关键词 |
|------|---------|-----------|
| 1 | 冒充公检法 | 涉嫌洗钱、拘捕令、资金核查 |
| 2 | 投资理财诈骗 | 高收益、保本、内幕消息 |
| 3 | 兼职刷单 | 点赞返利、足不出户、日结 |
| 4 | 虚假贷款 | 无抵押、低利率、快速放款 |
| 5 | 杀猪盘 | 恋爱、博彩、投资平台 |
| 6 | AI语音合成诈骗 | 子女声音、紧急汇款 |
| 7 | 视频深度伪造 | 熟人借钱、裸聊勒索 |
| 8 | 虚假征信 | 征信修复、逾期洗白 |
| 9 | 购物退款 | 质量问题、双倍赔偿 |
| 10 | 游戏交易 | 装备交易、账号买卖 |
| 11 | 追星诈骗 | 粉丝群、打榜、明星福利 |
| 12 | 医保诈骗 | 异地报销、冻结解冻 |

#### 3.2.4 个性化风险评估
```
用户画像维度:
- 基础属性: 年龄段、性别、职业
- 行为特征: 对话习惯、风险偏好
- 历史记录: 遭遇诈骗次数、预警响应
- 社交网络: 联系人活跃度

风险阈值动态调整:
- 老年人: 阈值降低50%，增加确认环节
- 未成年人: 所有转账相关高亮警示
- 财会人员: 涉及汇款多重验证
```

### 3.3 模块三：实时干预与交互

#### 3.3.1 分级预警机制
| 等级 | 触发条件 | 干预措施 | 响应时效 |
|------|---------|---------|---------|
| 🟢 安全 | 无风险特征 | 正常对话 | - |
| 🟡 关注 | 模糊风险信号 | 温和提醒 | 30分钟内 |
| 🟠 警告 | 中度风险特征 | 弹窗警告 | 即时 |
| 🔴 紧急 | 高风险/涉钱 | 语音阻断+通知监护人 | <10秒 |

#### 3.3.2 监护人联动
- 紧急联系人一键添加
- 风险事件即时推送（短信/微信）
- 远程干预请求通道
- 监护人端查看被监护人安全报告

#### 3.3.3 报告生成
- 《安全监测日报/周报》
- 《风险事件详情报告》
- 《防护建议报告》
- 可视化图表：风险趋势、诈骗类型分布

### 3.4 模块四：自适应进化模块

#### 3.4.1 案例自动导入流程
```
新案例发现 → 数据清洗 → 格式标准化 → 向量化入库 → 知识库更新
```

#### 3.4.2 增量学习机制
- 每周自动更新热点诈骗案例
- 模型微调基于新样本
- 提示词动态优化

## 8. 关键技术实现细节

### 8.1 Agent核心实现 (基于LangGraph)

```python
# Agent状态机
states = {
    "idle": "等待输入",
    "receiving": "接收多模态数据",
    "analyzing": "分析中",
    "reasoning": "推理决策",
    "warning": "预警干预",
    "evolving": "知识更新"
}
```

### 8.2 多模态融合策略
```
文本特征 → BERT/Embedding
音频特征 → Whisper + MFCC
图像特征 → CLIP/ViT

特征拼接 → Cross-Attention → 融合向量 → 风险分类
```

### 8.3 RAG检索增强
```
用户输入 → Embedding → 向量检索 → Top-K案例 → 上下文增强 → LLM生成
```
