# SmartGuard 智能反诈助手

## 产品演示指南

---

## 一、项目简介

SmartGuard 是一款**人工智能驱动的多模态反诈智能助手**，帮助用户识别诈骗信息、保护家人安全。

### 核心价值

- 🔒 **智能识别** - AI精准识别12+种诈骗类型
- 👨‍👩‍👧 **家人守护** - 紧急情况自动通知监护人
- 🧠 **持续学习** - 从案例中不断进化防护能力
- 📱 **多端覆盖** - Web、手机App、邮件监控全覆盖

---

## 二、为什么需要 SmartGuard？

### 触目惊心的数据

| 数据 | 说明 |
|------|------|
| **4700亿+** | 2023年全国电信网络诈骗损失金额 |


### 功能一：多模态输入

### 功能二：智能风险评估

SmartGuard 系统采用 **5级风险评估体系**（风险等级 0-4），精准判断内容安全性：

```
┌────────────────────────────────────────────────────────┐
│                    风险等级说明                        │
├────────────────────────────────────────────────────────┤
│ 🟢 0级  │ 安全    │ 内容完全可信，正常通过             │
│ 🔵 1级  │ 低风险  │ 有轻微可疑，注意即可               │
│ 🟡 2级  │ 中风险  │ 存在风险，建议警惕                 │
│ 🟠 3级  │ 高风险  │ 很可能是诈骗，请谨慎处理           │
│ 🔴 4级  │ 极高危  │ 确认诈骗！立即停止操作 + 通知监护人│
└────────────────────────────────────────────────────────┘
```

> **说明**：代码实现中等级4为最高风险，会触发监护人紧急通知。系统实际采用0-4共5个风险等级。

**演示方式：** 用测试案例展示不同风险等级的响应

## 五、实际演示案例

### 演示案例一：短信诈骗识别

**场景：** 用户收到一条可疑短信

```
📱 收到短信：
"【社保局】您的社保卡已停用，请点击链接激活：
http://fake-verify.com"
```

**演示步骤：**
1. 用户打开 SmartGuard
2. 切换到「风险分析」模式
3. 粘贴短信内容
4. 点击发送
5. 系统返回分析结果

**预期结果：**
```
⚠️ 这是一条钓鱼短信！

【风险分析】
• 冒用官方机构（社保局）名义
• 包含可疑短链接
• 诱导点击外部链接

【风险等级】🔴 4级 (极高危)

【建议】
1. 不要点击任何链接
2. 不要填写任何信息
3. 可向12333官方核实
4. 建议举报该短信
```

### 演示案例二：咨询学习

**场景：** 用户想了解"杀猪盘"诈骗

```
用户：什么是杀猪盘？
```

**系统响应：**
```
🛡️ 让我来为你详细介绍「杀猪盘诈骗」：

【什么是杀猪盘？】
「杀猪盘」是一种网络交友诈骗，骗子通过婚恋网站
或社交平台寻找目标，通过长时间聊天建立感情（养猪），
最后诱导受害者参与虚假投资（杀猪）。

【典型特征】
• 通过婚恋网站主动搭讪
• 聊天几天就建立恋爱关系
• 频繁提到投资、博彩、平台
• 诱导下载未知投资App

【真实案例】
王女士在网上认识一名"成功人士"，被骗28万元...

【防范要点】
✅ 网恋不见面的都是诈骗
✅ 对方提到投资立刻警惕
✅ 不下载来历不明的App
```
