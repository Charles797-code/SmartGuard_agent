## 1. 项目概述

### 1.1 项目背景

随着诈骗案件的高发，全民反诈已成为国家和社会的重要任务。本项目基于大语言模型（LLM）和多模态大模型（VLM），设计并开发了一个具备"感知-决策-干预-进化"能力的智能助手。

### 1.2 项目目标

- 提供多模态输入支持（文本、语音、图像、视频）
- 实现智能识别与决策引擎
- 提供实时干预与交互能力
- 构建自适应进化模块
- **支持邮件监控，自动检测邮箱中的可疑邮件**

### 1.4 技术亮点与创新

本项目在反欺诈智能助手领域具有多项技术创新和独特设计，主要体现在以下几个方面：

#### 1.4.1 多模态感知架构

本系统支持 **文本(Text)、音频(Audio)、图像(Image)、视频(Video)** 四种模态的输入融合分析，采用加权融合、注意力融合、投票融合三种策略，实现跨模态风险信号的交叉验证与综合研判。

| 融合策略 | 原理 | 适用场景 |
|---------|------|---------|
| 加权平均融合 | 按各模态置信度加权计算综合风险分 | 通用场景，默认策略 |
| 注意力融合 | 引入注意力机制动态调整权重 | 多模态信号权重差异大 |
| 投票融合 | 各模态独立判断后投票表决 | 需要保守稳健的判断 |

**代码位置**：`agent/src/modules/recognizer/fusion.py`

```python
def _weighted_average_fusion(self, features, weights):
    """加权平均融合"""
    weighted_sum = 0.0
    weight_total = 0.0
    for modality, feature in features:
        w = weights.get(modality, 1/len(features))
        confidence_weight = feature.confidence
        weighted_sum += feature.risk_score * w * confidence_weight
        weight_total += w * confidence_weight
    return weighted_sum / weight_total if weight_total != 0 else 0.5
```

#### 1.4.4 精细化关键词权重系统

区别于传统的关键词匹配，本系统设计了 **50+关键词精细化权重体系**，根据诈骗类型和危害程度动态调整风险权重：

| 风险等级 | 代表关键词 | 权重值 |
|---------|---------|-------|
| 极高危 | 安全账户、绑架 | 3.0 |
| 高危 | 拘捕令、洗钱、密码、验证码 | 2.5 |
| 中危 | 洗钱、紧急、汇款 | 2.0 |
| 一般 | 高收益、保本、稳赚不赔 | 1.5 |

**代码位置**：`agent/src/core/decision.py:69-150`

### 6.2 核心模块设计

#### 6.2.1 Agent核心模块

```python
class AntiFraudAgent:
    """多模态反诈智能体"""
    
    def __init__(self, user_id, llm_client, knowledge_base_path, ...):
        self.user_id = user_id
        self.llm_client = llm_client
        self.memory = UserMemory(user_id)
        self.knowledge_retriever = KnowledgeRetriever(knowledge_base_path)
        self.alert_manager = AlertManager()
        self.guardian_notifier = GuardianNotifier()
        
    async def process(self, input_data: AgentInput) -> AgentOutput:
        # 1. 感知：多模态输入处理
        perceived = await self._perceive(input_data)
        
        # 2. 决策：风险评估
        decision = await self._decide(perceived)
        
        # 3. 干预：执行预警
        await self._intervene(decision)
        
        # 4. 进化：学习新知识
        await self._evolve(decision)
        
        return decision
```

#### 6.2.2 意图识别模块

```python
class IntentRecognizer:
    """意图识别器"""
    
    async def recognize(self, text: str, context: Dict) -> IntentResult:
        # 使用Prompt Engineering进行意图识别
        prompt = self._build_intent_prompt(text, context)
        result = await self.llm_client.chat([{"role": "user", "content": prompt}])
        return self._parse_intent_result(result)
```

#### 6.2.3 风险评估模块

```python
class RiskEvaluator:
    """风险评估器"""
    
    async def evaluate(self, input_data, user_profile) -> RiskAssessment:
        # 1. 关键词检测
        keyword_level = self._keyword_check(input_data.text)
        
        # 2. 案例匹配
        similar_cases = await self._match_cases(input_data.text)
        
        # 3. LLM分析
        llm_analysis = await self._llm_analyze(input_data, user_profile)
        
        # 4. 综合评分
        final_score = self._calculate_score(keyword_level, similar_cases, llm_analysis)
        
        return RiskAssessment(
            risk_level=final_score.level,
            risk_type=final_score.type,
            confidence=final_score.confidence,
            analysis=final_score.analysis
        )
```
